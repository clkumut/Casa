# Gate Evidence Records

## Amac

Bu klasor, Casa V1 gate'leri icin toplanan kanit paketlerini resmi EVD kaydi olarak tutar. Bir gate icin kanit dosyasi acilmasi, gate'in gectigi anlamina gelmez.

## Kullanim Standardi

- Dosya adlari `EVD-Gx-xxx-kisa-baslik.md` seklindedir.
- Her kayit ilgili ORD, work package ve APR kaydina baglanir.
- `Draft - Evidence Collected` durumu, kanitlarin derlendigi fakat resmi onayin alinmadigi anlamina gelir.
- `Draft - Planned Evidence` veya `Not Started` durumlari, gerekli kanit setinin henuz toplanmadigi gate'ler icin kullanilir.

## Minimum Icerik

- Kimlik ve bagli gate bilgisi
- Gercek durum ozeti
- Toplanan kanitlar veya planlanan kanit listesi
- Acik eksikler ve sonraki adim
