# Gate Approval Records

## Amac

Bu klasor, gate kapanislari icin gerekli resmi sign-off kayitlarini tutar. Approval dosyasinin acilmasi, onay verildigi anlamina gelmez.

## Kullanim Standardi

- Dosya adlari `APR-Gx-xxx-kisa-baslik.md` seklindedir.
- Her approval kaydi ilgili EVD dosyasina baglanir.
- `Pending` durumu, kanit mevcut olsa bile karar verilmedigi anlamina gelir.
- `Draft` veya `Not Started` durumlari, approval toplama surecinin henuz baslamadigi gate'ler icin kullanilir.

## Minimum Icerik

- Kimlik ve bagli gate bilgisi
- Gerekli imza rolleri
- Mevcut karar durumu
- Acik kosullar ve sonraki adim
