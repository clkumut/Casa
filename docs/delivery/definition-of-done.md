# Definition of Done

## Amaç

Bir work package veya issue'nin tamamlandigi ancak asagidaki kosullar saglandiginda kabul edilir.

## Done Kriterleri

1. Kod veya konfigurasyon ilgili dokumana uygun uygulanmis olmali.
2. Ilgili test, emulator dogrulamasi veya smoke kaydi gecmis olmali.
3. Mock runtime veri veya placeholder logic kalmamis olmali.
4. Gerekli ise docs, EVD ve APR baglantilari guncellenmis olmali.
5. Security ve trusted write ihlali bulunmamali.
6. RTL/Arapca dogrulamasi gereken yuzeylerde sonuc kaydi olmali.
7. PM veya gate owner kapanis kaydini kabul etmis olmali.
