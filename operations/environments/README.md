# Environments

Bu klasor, local emulator haricindeki dev, staging ve prod ortam ailelerine ait operasyonel notlari ve baglam klasorlerini toplar.

Bu alana ortam bazli sorumluluk aciklamalari ve kontrollu operasyon artefact'lari girer. Hardcoded proje id, secret veya deploy hedeflerini dogrudan sabitleyen dosyalar burada tutulmaz.

Ana referans: `docs/ops/firebase-environments.md`
