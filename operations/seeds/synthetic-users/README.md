# Synthetic Users

Bu klasor, pseudonymous test kullanicilari ve bunlara ait seed veri ailelerini toplar.

Bu alana gercek kisi icermeyen, tutarli progression ve ekonomi durumuna sahip test kullanici veri setleri girer. Production PII, gercek e-posta adresleri veya birebir kullanici kopyalari burada tutulmaz.

Ana referanslar: `docs/ops/seed-data-strategy.md`, `docs/ops/firebase-environments.md`
