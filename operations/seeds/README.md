# Seeds

Bu klasor, emulator ve non-prod ortamlar icin kullanilan onayli seed veri ailelerini toplar.

Bu alana manifest'ler, canonical content setleri, system baseline ve synthetic kullanici veri aileleri girer. Runtime placeholder JSON'lar, production veri kopyalari veya UI'yi kandiran gecici listeler burada tutulmaz.

Ana referanslar: `docs/ops/seed-data-strategy.md`, `docs/ops/emulator-strategy.md`
