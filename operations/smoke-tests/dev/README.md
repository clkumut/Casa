# Dev Smoke Tests

Bu klasor, dev ortaminda veya emulator destekli akislar icin kullanilan smoke checklist baglamini toplar.

Bu alana hizli entegrasyon dogrulamalari ve dev release adayina ait kayitlar girer. Production kabul kaniti veya ortama ait gizli ayarlar burada tutulmaz.

Ana referanslar: `docs/runbooks/smoke-tests.md`, `docs/ops/emulator-strategy.md`
