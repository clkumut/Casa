# Incident Runbook Alanı

Bu klasor, deploy veya release sonrasi olay yonetimi baglamini desteklemek icin ayrilmistir.

Bu alana incident triage, iletişim ve ilk mudahale yardimci artefact'lari girer. Kalici runtime kodu veya security onayi gerektiren hassas log dump'lari burada tutulmaz.

Ana referanslar: `docs/runbooks/rollback.md`, `docs/ops/operational-readiness.md`
