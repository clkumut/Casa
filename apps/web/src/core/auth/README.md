# Core Auth

Bu klasor, istemci tarafindaki kimlik dogrulama baglami ve auth entegrasyonlarini barindirir.

Bu alana auth session adaptasyonlari, claim yorumlama ve auth akisini destekleyen uygulama cekirdegi girer. Feature-ozel izin kurallari veya backend trusted write kontrolleri burada tutulmaz.
