# Core State

Bu klasor, auth, shell UI ve app-level meta-state gibi global tutulmasi gereken istemci durumunu barindirir.

Bu alana cekirdek state store'lari ve app-level signal veya stream adaptasyonlari girer. Feature-local form state ve domain projection cache'leri burada biriktirilmez.
