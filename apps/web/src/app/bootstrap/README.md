# Bootstrap

Bu klasor, Angular uygulamasinin ilk ayaga kalkis sirasinda gereken bootstrap kompozisyonunu toplar.

Bu alana provider kayitlari, app-level initialization ve startup wiring girer. Feature mantigi, ekran seviyesinde route davranisi veya veri repository implementasyonlari burada tutulmaz.

Ana referans: `docs/architecture/angular-application-architecture.md`
