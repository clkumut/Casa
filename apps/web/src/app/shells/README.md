# Shells

Bu klasor, PublicShell, AuthOnboardingShell, AppShell ve OpsShell gibi uygulama kabuklarini barindirir.

Bu alana shell layout kontratlari ve kabuk seviyesindeki kompozisyon girer. Feature-ozel ekranlar, repository'ler veya trusted backend komutlari burada yer almaz.
