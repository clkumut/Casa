# App Katmani

Bu klasor, istemcinin uygulama seviyesindeki baslatma akisini, shell secimini ve route ailelerini tanimlamak icin ayrilmistir.

Bu alana uygulama kompozisyonu, route kaydi ve bootstrap koordinasyonu girer. Domain is kurallari, veri erisim adapter'lari veya feature-ozel UI bileşenleri burada tutulmaz.
