# Functions

Bu klasor, Firebase Cloud Functions tabanli trusted backend katmanini barindirir.

Bu alana callable command handler'lari, event processor'lar, scheduler akislari ve ops odakli trusted write yuzeyleri girer. Istemci UI kodu, docs artefact'lari veya ortama ozel gizli bilgi burada tutulmaz.
