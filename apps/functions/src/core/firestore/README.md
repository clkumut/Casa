# Core Firestore

Bu klasor, functions tarafinda Firestore ile kurulan cekirdek baglanti ve ortak veri erisim yordamlarini barindirir.

Bu alana transaction yardimcilari, collection referans factory'leri ve projection/update altyapisi girer. Domain repository davranisi veya UI projection varsayimlari burada tutulmaz.
