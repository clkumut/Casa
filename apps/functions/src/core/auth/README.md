# Core Auth

Bu klasor, callable ve event tabanli trusted write akislarinda kimlik ve claim dogrulama yardimcilarini toplar.

Bu alana auth context cozumleme, rol kontrolu ve yetki dogrulama yardimcilari girer. Domain-level is kararlari veya istemci session kodu burada tutulmaz.
