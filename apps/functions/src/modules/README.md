# Modules

Bu klasor, trusted backend domain modullerinin ana girisidir.

Bu alana her domain icin `application`, `domain`, `infrastructure`, `models` ve `handlers` aileleriyle ayrilan moduller girer. Cross-cutting core servisler veya runtime bootstrap baglantilari burada biriktirilmez.

Ana referanslar: `docs/architecture/repo-and-folder-standards.md`, `docs/architecture/firebase-platform-architecture.md`
